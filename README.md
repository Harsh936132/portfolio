## 🚀 Features
- **Modern UI/UX**: Sleek dark-mode aesthetic with vibrant cyan/blue gradients and glassmorphism elements.
- **Fully Responsive**: Implemented responsive typography (`clamp()`), mobile-friendly flex stacking, and a functional Hamburger navigation menu for flawless rendering on all devices.
- **Dynamic Projects**: Custom grid layout that prominently highlights featured applications.
- **Direct Contact Links**: Integrated quick access to my LinkedIn, GitHub, Email, and Phone number.

## 🛠️ Built With
- **[React](https://reactjs.org/)** - Component-based architecture.
- **[Vite](https://vitejs.dev/)** - Next Generation Frontend Tooling for fast builds.
- **CSS3** - Built with custom vanilla CSS and CSS variables to handle the design system without heavy frameworks.

## 📥 Local Development

1. Clone the repository
```bash
git clone https://github.com/Harsh936132/portfolio.git
```
2. Navigate into the directory
```bash
cd portfolio
```
3. Install dependencies
```bash
npm install
```
4. Start the development server
```bash
npm run dev
```
